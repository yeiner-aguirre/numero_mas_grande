¡Formador de Números Más Grandes!

¿Qué hace mi programa?
------------------------
Este programa toma una serie de números que tu le das y los reorganiza para crear el numero más grande posible.

pasos para formar el numero mas grande posible
----------------
- Solo puedes usar numeros. Nada de letras, o simbolos raros
- Separa tus números con comas.
- No importa si dejas espacios entre los números, al programa no le importara.
- Puedes usar números de cualquier tamaño.

Ejemplos prácticos
------------------
Entrada: 50,2,1,9
Resultado: 95021

Entrada: 420,42,4
Resultado: 44242

Entrada: 1000,1,10,100
Resultado: 110100010

NOTA
-----------------
Lo mas importante es la funcion la cual la cree en javaScript y donde se implementa toda la logica de mi programa.
